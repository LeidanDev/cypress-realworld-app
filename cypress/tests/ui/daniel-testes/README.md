# Testes de Automação – Daniel

## 📌 Descrição

Esta pasta contém os testes de automação desenvolvidos por Daniel para o projeto **Real World App**.  
Os testes cobrem cenários de transferência de dinheiro, validação de saldo, mensagens de sucesso e erro.

---

## 🛠️ Pré-requisitos

Antes de rodar os testes, certifique-se de ter instalado:

1. **Node.js** (recomendado 18+):  
   [https://nodejs.org/](https://nodejs.org/)

2. **Gerenciador de pacotes**: Yarn ou npm

3. **Dependências do projeto**: na raiz do projeto execute:

```bash
yarn install
# ou
npm install
```

## Rodar o Front End

4. Frontend rodando: os testes de UI dependem da aplicação estar disponível. Execute:

yarn dev
# ou
npm run dev

##  Como rodar os testes

# Modo interativo (Cypress Test Runner)

5. npx cypress open

# Modo headless (linha de comando)

6. npm run test:daniel
